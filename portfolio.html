ATTETARKIAINEN.COM

HTML / CSS / PHP

Resume & portfolio website designed, coded & implemented by me. This project enabled me to bring back to mind the HTML & CSS languages, use the Bootstrap CSS & JavaScript library & get my feet wet with PHP by creating a validation script to handle the email contact form on the web page.

The technologies I used during this project: HTML, CSS, Bootstrap, JavaScript, PHP & GIT via GitHub for version control.

CONTACT_BOOK.PY

Python / SQL

The goal of this project was to expand my knowledge of the class structures in Python & create an application with multiple classes that rely on each other to function. The contact book uses the PyQT -library to create a functional & simple GUI, SQLite3 -library to create and manage an SQL database of the contacts & the smtplib & ssl -libraries to connect to a SMTP server securely & send customized emails to the selected contacts.

The technologies I used during this project: Python, PyQT, SQLite3, SMTP & GIT via GitHub for version control. This project is still a work in progress.

POSTILAATIKOT.FI & POSTBOXESUKLTD.CO.UK

HTML / CSS / WordPress

I was involved in building the PostBoxes UK LTD's Finnish website (postilaatikot.fi) & in updating their English website (postboxesukltd.co.uk). CALCULATOR.PY

Python

Fully functional calculator with a GUI. This was my first ever Python project & I set myself the goals of learning the syntax, learning how to use a GUI library to create a simple GUI & how to use classes & functions to create an easily readable & efficient code structure. I chose to use Tkinter as the GUI library for the sole reason of it being packaged with Python & I didn't want any external dependencies on the project. The Calculator´s functionalities and GUI layout were inspired by the standard Windows calculator.

The technologies I used during this project: Python, Tkinter & GIT via GitHub for version control.
